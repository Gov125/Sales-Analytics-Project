# Sales Analytics Project

## Project Overview

This project analyzes sales data using Python and Power BI to uncover business insights, customer trends, and sales performance.

## Tools Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook
* Power BI

## Dataset

The project uses two datasets:

* Orders.csv
* Details.csv

These datasets were merged and cleaned before analysis.

## Data Cleaning

* Checked missing values
* Verified data types
* Converted Order Date to datetime format
* Checked duplicate records
* Reviewed categorical values
* Validated numerical columns

## Exploratory Data Analysis (EDA)

The analysis includes:

* Monthly Sales Trends
* Category Performance
* Customer Analysis
* State-wise Sales Analysis
* Payment Mode Analysis
* Profit Analysis

## Key Insights

* January recorded the highest sales.
* July recorded the lowest sales.
* Sales recovered significantly after July.
* November showed strong sales performance.
* Category and customer analysis identified major revenue contributors.

## Dashboard

An interactive Power BI dashboard was created to visualize sales performance, customer behavior, and business metrics.
